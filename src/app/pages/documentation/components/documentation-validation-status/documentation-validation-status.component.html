Validation Status

The Validation status API let you to check the validation status of a request.

Location: GET https://api.checkmobi.com/v1/validation/status/&213;id&215;

Arguments

None

Request

Method: GET
URI: https://api.checkmobi.com/v1/validation/status/&213;id&215;
HTTP headers: Content-Type, Authorization. More details here
Body: N/A
CURL Example:

Response

HTTP status code: 200

Response Attributes

Parameter Type Description
number string The number that needs to be validated
validated boolean true if the pin was correct. false Otherwise.
validation_date integer The date time as UNIX timestamp when the validation was completed (the pin was matched first time). In case the number is not validated the value is null
charged_amount float The amount charged for that validation. For Reverse CLI (Missed call) only in case the user is fast enough to answer the call you are charged for 1 minute on that destination.
