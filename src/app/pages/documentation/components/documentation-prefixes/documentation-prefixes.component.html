Get Prefixes

This API allow you to get a list of all available countries and the associated calling prefixes.

Location: GET https://api.checkmobi.com/v1/prefixes

Request

Method: GET
URI: https://api.checkmobi.com/v1/prefixes
HTTP headers: Content-Type, Authorization. More details here.
Body: N/A
CURL Example:

Response

HTTP status code: 200

Response Attributes

A list of items. Each item will have the following properties:

Parameter Type Description
iso2 string Country ISO code 2 digits
name string Country name
prefix string Calling prefix for that country
