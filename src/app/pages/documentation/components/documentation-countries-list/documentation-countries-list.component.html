Get Countries

This API allow you to get a list of all available countries, flags and country code.

Location: GET https://api.checkmobi.com/v1/countries

Request

Method: GET
URI: https://api.checkmobi.com/v1/countries
HTTP headers: Content-Type, Authorization. More details here.
Body: N/A
CURL Example:

Response

HTTP status code: 200

Response Attributes

A list of items. Each item will have the following properties:

Parameter Type Description
iso2 string Country ISO code 2 digits
name string Country name
prefix string Country calling code
flag_32 string URL to the flag location. Size 32x32 pixels
flag_128 string URL to the flag location. Size 128x128 pixels
