Call Status

The Call status API let you to check the status of a call request.

Location: GET https://api.checkmobi.com/v1/call/&213;id&215;

Arguments

None

Request

Method: GET
URI: https://api.checkmobi.com/v1/call/&213;id&215;
HTTP headers: Content-Type, Authorization. More details here
Body: N/A
CURL Example:

Response

HTTP status code: 200

Response Attributes

Parameter Type Description
request_date integer The date time as UNIX timestamp when the request was placed
to string The destination number
description string A short description about carrier and country: for example: Orange - FR
duration_billed integer Represent the length of the call in seconds between answer and hangup
charged_amount float Total cost for the call based on charged_rate and duration
charged_rate float Charged rate per destination
