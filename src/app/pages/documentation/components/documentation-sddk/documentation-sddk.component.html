SDK Overview

CheckMobi SDKs let you integrate the validation API in the browser and mobile devices.

Android SDK

  • Remote Config SDK for Android (recommended)
  • Android SDK (deprecated)

iOS SDK

  • Remote Config SDK for iOS (recommended)
  • Objective C SDK for iOS (deprecated)
  • Swift SDK for iOS (deprecated)
  • Swift - Missed Call (community contribution)

PHP SDK

  • PHP SDK
  • PHP Laravel SDK (community contribution)

Node.js SDK

  • omrs-checkmobi (community contribution)

Ruby SDK

  • mithucste30/check_mobi (community contribution)
  • HEXA-UA/checkmobi-sdk (community contribution)

Other platforms or programming languages

For all the other platforms/programming languages you can use the REST API we support in order to use our product.
