Get Account details

Using the account API you can retrieve your account details like email, available cash credits, expiration date, monthly usage, etc.

Location: GET https://api.checkmobi.com/v1/my-account

Request

Method: GET
URI: https://api.checkmobi.com/v1/my-account
HTTP headers: Content-Type, Authorization. More details here.
Body: N/A
CURL Example:

Response

HTTP status code: 200

Response Attributes

Parameter Type Description
email string The account associated email
subscription array The subscription package details associated with the accounts (id, name, how many cli and missed call validations are included)
cli_validations integer How many CLI and Missed Call (Reverse CLI) validations were performed
sms_validations integer How many SMS validations were performed in the last month
ivr_validations integer How many IVR validations were performed in the last month
sms_api integer How many SMS's were sent through SMS API in the last month
voice_api integer How many calls were sent through Voice API
expiration_date integer The UNIX timestamp in seconds when the subscription will expire
credit_balance float The current account balance
